# LF-mc QEST+FORMATS 2026 Artifact Page

This repository contains the static GitHub Pages site for the artifact associated
with the QEST+FORMATS 2026 tool paper "LF-mc: An Efficient Verifier for Lingua
Franca".

Open `index.html` locally, or publish the repository root with GitHub Pages.
The Docker image is hosted on GitHub Container Registry as:

```sh
ghcr.io/symbolicsafety/lf-mc:v1.2
```

The detailed artifact instructions are in `artifact/README.html`.
