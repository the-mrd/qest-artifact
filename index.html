LF-mc: An Efficient Verifier for Lingua Franca

Welcome to the artifact page for our QEST+FORMATS 2026 tool paper, "LF-mc: An Efficient Verifier for Lingua Franca".

This page provides the paper and accompanying artifact prepared to support the paper's evaluation. It contains:

  1. A PDF of the submitted paper.
  2. A Docker-based environment for running LF-mc.
  3. An script for reproducing the benchmark results from the paper.
  4. The source archive containing the lf-maude and lf-mc source trees of the tool.
