LF-mc: An Efficient Verifier for Lingua Franca

LF-mc is a verification tool for Lingua Franca programs. It compiles annotated LF models to Maude, runs simulation, reachability, and LTL analyses, and reports the results through the LF toolchain.

Quick Start

The normal reviewer path uses the Docker image hosted on GitHub Container Registry. The repository does not host Docker image archives.

user@host:~$ docker pull ghcr.io/symbolicsafety/lf-mc:v1.2
user@host:~$ docker run -it --rm ghcr.io/symbolicsafety/lf-mc:v1.2 bash -l
docker:/# lfc /opt/lf-maude/examples/src/ADASModel.lf

Replicating Table 2

From the artifact directory, run the provided script. It starts one Docker container, invokes the bundled hyperfine, runs each benchmark listed in tests.csv, and writes results.csv, hyperfine-report.json, and per-benchmark logs.

user@host:repo$ cd artifact && sh run-all-experiments.sh --overwrite
