LF-mc: An Efficient Verifier for Lingua Franca

Welcome to the artifact page for our QEST+FORMATS 2026 tool paper, "LF-mc: An Efficient Verifier for Lingua Franca".

This page provides the paper and accompanying artifact prepared to support the paper's evaluation. It contains:

  1. A PDF of the submitted paper.
  2. A Docker-based environment for running LF-mc.
  3. The LF-mc v1.2 package for use without building from source.
  4. An experiment archive for reproducing the LF-mc column of Table 2.
  5. The list of 22 LF-Verifier benchmark programs used in the evaluation.
  6. The source archive containing the lf-maude and lf-mc source trees.
  7. Instructions for running the tool, reproducing the experiments, and building from source.
